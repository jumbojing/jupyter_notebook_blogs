docker命令

查看本地镜像

~> docker images                            
REPOSITORY            TAG       IMAGE ID       CREATED       SIZE
fastdotai/nbdev-dev   latest    f37f83e5f8a6   3 weeks ago   2.11GB

建镜像并挂本地目录

~> docker run -it -v /Users/liguimei/Documents/dockerlocal:/home/local fastdotai/nbdev-dev /bin/bash
